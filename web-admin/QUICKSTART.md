# 🚀 快速启动指南

## 一、环境准备

1. 确保已安装 Node.js (版本 >= 16)
2. 确保后端服务已启动在 `http://localhost:8080`

## 二、安装与运行

```powershell
# 进入项目目录
cd web-admin

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 `http://localhost:3000` 即可看到登录页面

## 三、登录系统

开发环境默认凭据：
- 手机号：13800000000（或任意有效手机号）
- 验证码：123456

步骤：
1. 输入手机号
2. 点击"获取验证码"
3. 输入验证码 123456
4. 点击"登录"

## 四、功能导航

登录成功后，可以访问以下页面：

- **数据统计** (`/dashboard`) - 查看用户数据概览
- **用户管理** (`/users`) - 管理所有用户
- **系统设置** (`/system`) - 配置系统参数

## 五、常见问题

### Q: 无法启动？
A: 检查端口 3000 是否被占用，可以修改 `vite.config.js` 中的端口号

### Q: 登录失败？
A: 确保后端服务正常运行，验证码使用 123456

### Q: 接口报错？
A: 检查后端服务是否启动，查看浏览器控制台网络请求

## 六、后续开发
